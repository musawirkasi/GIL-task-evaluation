Components:

Arduino UNO
1602A LCD module
Rotary Encoder module
3 Relay modules

Connections:

Rotary Encoder:

CLK -> Pin 2
DT -> Pin 3
SW (button) -> Pin 4
VCC -> 5V
GND -> GND

LCD (I2C):

SDA -> A4
SCL -> A5
VCC -> 5V
GND -> GND

Relay Module:

Relay 1 IN -> Pin 8 (BULB 1 control)
Relay 2 IN -> Pin 9 (BULB 2 control)
Relay 3 IN -> Pin 10 (BULB 3 control)
VCC -> 5V
GND -> GND
